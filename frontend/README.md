# Farm Direct Frontend

React + Vite + TypeScript frontend for Farm Direct application.

## Installation

```bash
npm install
```

## Running Development Server

```bash
npm run dev
```

The app will run on `http://localhost:8080`

## Building for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite (build tool)
- Tailwind CSS
- Shadcn UI components
- React Router
- React Hook Form
- TanStack React Query
- Recharts (data visualization)

## Environment Configuration

Update API endpoints in your environment configuration to point to the backend:
- Backend URL: `http://localhost:4000`

## Features

- User authentication
- Role-based dashboards (Farmer, Buyer, Admin)
- Product marketplace
- Order management
- Real-time chat
- Ratings and reviews
- Location-based filtering
- Delivery tracking
